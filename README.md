# 2048 Project

A Python implementation of the 2048 game with core logic, UI, and AI agents.

## Features

- Core 2048 game logic
- Graphical user interface
- Random agent
- Heuristic agent
- DQN agent
- Training and testing modules

## Project Structure

```text
src/        source code
tests/      test files
docs/       project documents
img/        images or assets
logs/       runtime logs
